Jika sudah masuk ke tahap selanjutnya yaitu menjalankan script,jika di projek sebelumnya menggunakan salah satu fitur virtualbox yaitu shared folder kali ini saya akan menggunakan github
sebagai tempat menyimpan script yang nantinya akan digunakan untuk exec beberapa task,langsung saja masuk ke github masing-masing lalu pilih salah satu repository kosong atau bisa juga
membuat repository baru untuk scrip ini jika sudah buat file dan isi file dengan shell ini

```
#!/bin/bash

useradd -m -s /bin/bash vagrant
echo "vagrant ALL=(ALL) NOPASSWD:ALL" > /etc/sudoers.d/vagrant

mkdir -p /home/vagrant/.ssh
curl -o /home/vagrant/.ssh/authorized_keys https://raw.githubusercontent.com/hashicorp/vagrant/main/keys/vagrant.pub

chown -R vagrant:vagrant /home/vagrant/.ssh
chmod 700 /home/vagrant/.ssh
chmod 600 /home/vagrant/.ssh/authorized_keys
```

jika sudah save dan pastika format save dengan .sh, jika sudah masuk ke vm yang tadi sudah di buat lalu git clone repo yang tadi berisi script sh,

```
git clone http://namarepo
```

jika sudah masuk kedalam repo yang baru saja di clone lalu run script tadi sebagai admin,tunggu hingga proses selesai lalu jika sudah selesai gunakan command


```
sudo apt clean
sudo apt autoremove -y
```

lalu gunakan command ini juga 

```
sudo truncate -s 0 /var/log/*.lo
```

agar nantinya proses installasi menjadi clean install dan bagus untuk golden image,sudah selesai untuk setup golden image jangan lupa hapus folder dan file yang tadi sudah di clone
masuk ke tahap selanjutnya yaitu provisioning vagrant untuk setup cluster,jika tadi golden image sudah di buat maka sekarang masuk ke pembuatan vm tapi sebelum itu harus memastikan
bahwa golden image bisa digunakan, gunakan command

```
vagrant package --base nama-vm-yang-sudah-dibuat --output golden.box
|
vagrant package --base node-1 --output golden.box
```

jika sudah maka vagrant siap untuk menggunakan golden image yang baru saja di tambahkan, langsung saja buat file di vscode dengan nama vgarant
