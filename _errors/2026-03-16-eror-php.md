### Overview

Pada post eror kali ini khusus membahas eror yang saya temukan pada saat melakukan deployment di post [ini](https://rizkiyusupk.github.io/devops/kubernetes/linux/server/deploy/deployment/deploy-php1/)

### Tidak bisa membuat image 

Dikarenakan di post deploy php dibutuhkan sebuah image untuk deploy dan imagenya itu di buat manual dan tidak menggunakan image dari registry
maka perlu untuk membuild semua dari awal nah proses build image ini sangat di perlukan karena nantinya kita akan membutuhkan imagenya untuk deploy,
dalam kasus ini prose ketika build itu tidak bisa di lakukan karena adanya eror yang terjadi ketika membuild image, saya akan contohkan seperti ini

```
docker build -t image .
|
docker build -t php-test2 .
```

nah saya akan coba command diatas untuk test build imagenya 

![asdhausd](/assets/images/php1/Screenshot 2026-03-16 132551.png)

nah contoh diatas bisa terjadi karena satu hal yaitu docker engine tidak berjalan,maka dari itu pastikan docker desktop menyala atau jika belum terinstall pastika install terlebih dahulu,
